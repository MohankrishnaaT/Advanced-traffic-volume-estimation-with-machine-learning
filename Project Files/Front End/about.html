üß† Model Information

Random Forest Regression Model

Our traffic prediction model is built using Random Forest Regression, chosen for its superior performance in handling complex, non-linear relationships in traffic data.

  • Algorithm: Random Forest Regressor
  • Feature Engineering: Date-time splitting, categorical encoding, feature scaling
  • Data Preprocessing: Missing value imputation, outlier detection, normalization
  • Model Evaluation: R¬≤ Score, RMSE metrics for performance assessment
  • Deployment: Serialized using Pickle for production deployment

üìä Input Parameters

The model considers the following factors for traffic volume prediction:

  • Holiday Status: Whether the day is a working day or holiday
  • Temperature: Ambient temperature in Celsius
  • Precipitation: Rainfall and snowfall measurements in millimeters
  • Weather Conditions: Overall weather description (Clear, Cloudy, Rainy, etc.)
  • Temporal Factors: Year, month, day, and hour for time-series analysis
